# Morning Koi Polished

A peaceful mobile-first idle fishing web app with a cozy pixel-art koi garden and lakeside cabin vibe.

## Added in this polished build
- richer fish motion and depth states
- rotating day-to-night sky moods
- weather shifts including breeze, mist, and soft rain
- rare catch celebration effects
- cleaner zen/screensaver mode
- custom pixel-art fish and environment assets

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder, keeping the folder structure intact.
3. In GitHub, go to Settings → Pages.
4. Set the source to Deploy from a branch.
5. Choose the main branch and the root folder.
6. Save, then wait for the site URL.

## Deploy to Netlify
1. Create a new Netlify site.
2. Drag and drop this whole folder, or connect a repo containing these files.
3. Publish the site.
